# Radius menu print layout

Страница сверстана под печать в формате **половина A4**: `105mm × 297mm`.

## Запуск

```bash
npm start
```

После запуска откройте `http://localhost:4173`.

## Как сохранить в PDF как половину A4

1. Откройте печать в браузере (`Cmd + P`).
2. Выберите:
   - Размер бумаги: `105 × 297 мм` (или custom size),
   - Масштаб: `100%`,
   - Поля: `None` / `Без полей`,
   - Background graphics: `On` / `Включено`.
3. Сохраните как PDF.

Файлы:
- `index.html` — структура меню и текст по окружностям,
- `styles.css` — размеры страницы, стили для печати, кольца.
